# cors-crud
<h3>simple node crud</h3>

# Step 1
clone application on your system
`git clone https://github.com/azadpunchy/cors-crud.git`

# Step 2
install dependencies 
`npm i`

# Step 3
set environment variables as
`DB=your mongodb url
`
`PORT= your port`
# Step 4
Start your server locally as
`node index.js`
